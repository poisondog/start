# start.swift

Create a package project
```bash
swift package init
```

Build this project
```bash
swift build
```

Test this project
```bash
swift test
```

Generate XCode project
```bash
swift package generate-xcodeproj
```

Run this project
```bash
swift run runStartSwift # write in Package.swift executable
```
